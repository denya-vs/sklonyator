Веб-сервис предназначен для склонения по падежам слов на русском языке.
API доступно по урлу {% url 'api' %}?s=замок&case=gent&number=plur и передаче 2-ух обязательных и 1-го необязательного GET-параметров:
  1. s=Слово которое необходимо просклонять
  2. case=падеж в котором нужно вернуть результат
    Возможны следующие параметры:
    • nomn - именительный (Кто? Что?)
    • gent - родительный (Кого? Чего?)
    • datv - дательный (Кому? Чему?)
    • accs - винительный (Кого? Что?)
    • ablt - творительный (Кем? Чем?)
    • loct - предложный (О ком? О чём?)
    • voct - звательный (Его формы используются при обращении к человеку).
  3. Необязательный - number=число(единственное - sing , множественное - plur)
